# cabalBot
A simple AHK AFK farming bot

;Requirements
;==================================================
AHK 1.1-* (AutoHotkey_L)

;Game requirements
;==================================================
1920 * 1080 in Windowed mode with tray on the bottom and 1 unit tall
Tahoma Bold 11.0 Font

;Installation
;==================================================
Put script anywhere then run it as admin
